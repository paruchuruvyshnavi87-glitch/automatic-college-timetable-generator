<u>Automatic College Timetable Generator</u>

A web-based application designed to automate college timetable generation with secure admin-controlled editing. This project focuses on solving real academic scheduling problems using core web technologies.

<u>Overview</u>

Manual timetable creation is time-consuming and error-prone. This project provides a simple, efficient, and customizable solution that allows institutions to generate and manage timetables dynamically with minimal input.

<u>Key Features</u>

Branch-wise timetable generation

Dynamic selection of periods per day

Automatic subject and lab allocation

Password-protected Admin Edit Mode

Role-based access for timetable editing

Editable timetable interface

PDF download support

Responsive and user-friendly UI
